# Clawd Spinner Words — Official Catalog

> Extracted from local reference screenshots and reconciled against `generate_clawd_gifs.py`. Cross-checked against the default `spinnerVerbs` table embedded in Claude Code `2.1.153` (187 verbs).

## Legend

- ✅ = GIF already generated (`generated/Clawd-{Word}.gif`)
- ⏳ = official spinner word, scene not yet drawn
- ★ = repo extension (not in current upstream defaults)
- "官方语义" gives a short English note on what the upstream verb is conveying to the user, when that differs from or is more specific than the Chinese scene description.
- `Clawd-Dancing.gif` is kept locally as an external visual reference and is not part of the official spinner-word catalog.
